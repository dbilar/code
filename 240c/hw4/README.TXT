Moves are in the following manner:
a5
g7
h4
etc...

If (and only if) a player has no legal moves, 
he/she is allowed to pass by typing pass.

The computer doesn't know the importance of corners yet,
so that's the easiest way to win.

The networking uses port 8888, so make sure it's not in use if you want to play 
a networked game.

To play a networked game where both terminals are on the same computer,
enter localhost as the IP address.
